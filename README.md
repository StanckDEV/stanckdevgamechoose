#  Stanck Game Choose
**La selezione definitiva dei migliori browser game per developer.**

Link al progetto: [stanckdevgamechoose.vercel.app](https://stanckdevgamechoose.vercel.app/)

---

##  Il Progetto
**Stanck Game Choose** non è la solita raccolta di giochi amatoriali. È un catalogo curato di web-games ad alte prestazioni (WebGL, 3D Physics) selezionati per chi cerca qualità tecnica anche nel browser.

Il sito segue la filosofia brutalista e minimale di **stanckDEV**: tipografia massiccia, gerarchia netta e massima velocità di caricamento.

##  Tech Stack
Per questo progetto ho puntato sulla leggerezza assoluta, garantendo un caricamento istantaneo su qualsiasi rete (anche quelle scolastiche/aziendali):

* **HTML5** - Struttura semantica e accessibile.
* **CSS3** - Layout moderno con Flexbox, CSS Grid e variabili per una gestione pulita dei colori.
* **JavaScript (Vanilla)** - Logica di *Scroll Spy* personalizzata per la navigazione dinamica nella sidebar.
* **Vercel** - Deployment e hosting ad alte prestazioni.

##  Caratteristiche principali
* **Design Responsive:** Ottimizzato per desktop e smartphone.
* **Sidebar Dinamica:** Il menu evidenzia automaticamente la sezione che stai visualizzando.
* **Selezione "High Performance":** Solo titoli con motori grafici avanzati (es. Rocket Goal, Shell Shockers).
* **Zero Bloatware:** Niente pubblicità invasiva, niente caricamenti infiniti. Solo il link e la descrizione.
